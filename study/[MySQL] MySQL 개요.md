# [**MySQL] MySQL 개요**

**MySQL**

MySQL은 가장 널리 사용되고 있는 관계형 데이터베이스 관리 시스템이다.

MySQL은 오픈 소스이며, 다중 사용자와 다중 스레드를 지원한다.
또한 C언어, C++, JAVA, PHP 등 여러 프로그래밍 언어를 위한 다양한 API를 제공하고 있다.
