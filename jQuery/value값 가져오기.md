## value값 가져오기

### 1. id로 접근해서 가져오기

→ var value = $(’**#**test_id’).val();

### 2. class로 접근해서 가져오기

→ var value = $(’**.**test_class’).val();

### 3. name으로 접근해서 가져오기

→ var value = $(’**input[name=test_name]**’).val();
